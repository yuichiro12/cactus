# cactus

An instant [hckrnews](https://hckrnews.com) radio.

## requirement
- [Goquery](https://github.com/PuerkitoBio/goquery) is required for get articles.
- To read text, `say` command is required.

## usage
```sh
go run main.go | say
```

## recommendation
Play with your own bgm and it's nicer.
