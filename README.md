# cactus

An instant [hckrnews](https://hckrnews.com) radio.

## requirement
- [Goquery](https://github.com/PuerkitoBio/goquery) is required for getting articles.
- To read text, `say` command is required.

## usage
```sh
go run main.go | say
```

## recommendation
With your own bgm, it will be nicer.
